A stock ticker tracking app that runs on desktop (Windows, MacOS, Linux)

# clone
`git clone https://github.com/tmazumdar/stocks.git`

# build
`yarn install`

# run
`yarn start`

# package
`yarn run make`
A distributable package will be created in `out` folder
